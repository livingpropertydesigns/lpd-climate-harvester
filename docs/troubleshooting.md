# Troubleshooting

## Common Issues

**"ZIP not found"**  
→ The ZIP code may not exist in the database. Try a nearby ZIP or contact us.

**Data shows "Fallback (default)"**  
→ This county has incomplete ASHRAE source data. The values shown are safe defaults.

**Manual J worksheet not pulling data**  
→ Check that:
- The **ZIP_Lookup** worksheet exists and is named exactly that
- Your database headers match your file’s layers and plugin styles

**Streamlit app not loading**  
→ Try refreshing the page or clearing your browser cache.
