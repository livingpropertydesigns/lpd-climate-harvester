# Contributing to Manual J Climate Harvester

Thank you for your interest in improving this tool! Contributions are welcome and appreciated.

## How to Contribute

1. **Fork the repository** and create a new branch for your changes.
2. **Make your changes** — keep code clean, well-commented, and consistent with the existing style.
3. **Test your changes** — especially any modifications to data processing or the UI.
4. **Submit a Pull Request** with a clear description of what you changed and why.

## Areas Where Help is Needed

- Improving data accuracy for additional states
- Enhancing the Streamlit UI
- Adding more shading/solar gain calculations
- Creating VectorScript or Marionette integration
- Improving fallback logic and documentation
- Writing tests

## Code Style

- Follow PEP 8 where possible
- Use clear variable names
- Add comments for complex logic
- Keep the UI simple and professional

## Questions?

Feel free to open an issue or reach out directly. We're building this together for the BIM community.

Thank you for helping make reliable climate tools more accessible!
